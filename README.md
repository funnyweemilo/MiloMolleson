# Milo Molleson — Portfolio Site

Personal art portfolio hosted on GitHub Pages.  
Live at: [https://MiloMolleson.github.io](https://funnyweemilo.github.io/MiloMolleson/index.html)

## Structure

```
index.html          # Homepage with image grid
exhibitions.html    # Exhibitions & previous work
images/             # All images go here
README.md
.gitignore
```

## Updating the site

**Add a new work to the homepage**  
Copy one of the `.work-item` blocks in `index.html` and update the `src` and caption.

**Add an exhibition**  
In `exhibitions.html`, copy an `.exh-item` block inside the right year group and fill in the title, venue, and type (Solo / Group / Print / Residency etc).

**Add a new year section**  
Copy a full `.exh-year-group` block and update the year label.

## Images

Put all images in the `images/` folder. Recommended:
- Hero image: `images/hero.jpg` (landscape, high resolution)
- Work images: `images/work1.jpg`, `work2.jpg` etc
- Exhibition docs: `images/exh1.jpg` etc

JPG is fine for photos/illustrations. Keep files under ~1MB where possible.
